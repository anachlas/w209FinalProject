Yesterday's News Today


The GDELT Project is a realtime network diagram and database of global human society for open research. Using the location data and news articles URLs we extract the key entities (Persons, Locations, Organizations) and explore their relationships.

Two Stages of Data Interaction

1. The map visualization allows the user to explore the volume of news from each country around the world and view example articles from each location.
2. Next to explore the relationships amongst the entities based on article co-occurrence a force directed graph has been built to highlight both notable entities (larger sized nodes) and closely related entities (more co-occurrences = stronger gravity). Clicking on the name of an entity on the graph opens up page of URLs that reference the selected entity.

By Thomas Atkins, Eric Whyne, and Alex Nachlas
